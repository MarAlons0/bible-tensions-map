About

An interactive tool for secular, academic analysis of the Bible as literature and historical document. It maps 65 biblical books along 20 analytical axes, asking: where does each book sit on each theological tension, and how do those positions shift across time and tradition?

View source on GitHub →

Views

Heatmap

The main view. Rows are books (or aggregates), columns are the 20 tensions, cells are colored blue (Pole A, −5) → gray (0) → coral (Pole B, +5). The heatmap is layered: it opens with Old Testament and New Testament aggregate rows showing median scores. Click a row to expand it into canonical sections, then again to expand sections into individual books. Click any cell in a book row to open a tension detail bar chart.

Biplot

A two-dimensional scatter plot. Select any two tensions as axes and see where each book falls. Useful for identifying clusters and outliers. Preset pairs highlight classic scholarly comparisons.

Conduct Codes

Tracks how nine categories of practical conduct are addressed across the canon: diet, marriage & sexuality, homosexuality, slavery, clothing, agriculture, economic justice, violence & war, and ritual calendar.

Timeline

A line chart showing how tension scores shift across books, ordered by scholarly dating (when scholars think the text was composed) or canonical order (the traditional sequence). Toggle between the two; select which tensions to overlay.

Chapter analysis

On any book page, individual chapters can be analyzed on demand via the Anthropic API (claude-sonnet-4-6). The model returns a summary, scores for all 20 tensions, and conduct code entries. Results are stored and reused on subsequent visits — no re-querying needed.

Data & methodology

65 books — 38 Old Testament + 27 New Testament. Scores are pre-computed scholarly assessments based on historical-critical consensus (drawing on resources such as the New Oxford Annotated Bible and standard academic commentaries). They reflect the dominant theological position of the text as reconstructed by scholars — not any single confessional tradition.

Dating follows critical consensus: e.g. Deuteronomy ~621 BCE, Second Isaiah ~550–540 BCE, Mark ~70 CE, Revelation ~95 CE. These dates drive the chronological ordering in the Timeline view.
